\# Backend Selector



Small Java practice project implementing round-robin backend selection.



The selector cycles through a list of backend servers and returns the next server for each request.



\## Features



\* Round-robin backend selection

\* Simple server list handling

\* Unit test with JUnit 5



\## Tech stack



\* Java

\* Maven

\* JUnit 5



\## Run tests



```cmd

mvn test

```



\## What this project demonstrates



\* basic load-balancing logic

\* state management

\* modular Java class design

\* unit testing with JUnit
